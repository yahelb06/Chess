The  Chess Project
In this exercise we implement classes that represent a full Chess Engine Backend using Object-Oriented Programming (OOP) principles. The engine contains core logic classes such as: Point ChessPiece (Abstract Base) Board King Queen Rook Bishop Knight Pawn When we finish implementing the logic and rule validation for all pieces, we will connect the engine to an external graphical frontend using a Pipe communication mechanism to run the game.
